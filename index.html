Tabs

La estructura principal sería algo así.
/ul.carrot-tabs li.active li ... /ul
Dentro agregar esto.
/div.carrot-content-tabs div.carrot-tab-item active h3.content-tab-title div.content-tab-body /div

Para la barra de navegación utiliza ul.carrot-tabs, mientras que para el contenido usa div.carrot-content-tabs. Puedes activar el tab con la clase active. Si lo que quieres es cambiar el color utiliza la clase .carrot-{color}. Por otro lado para activar y mostrar el tab actual es necesario agregar la clase active a div.carrot-tab-item
